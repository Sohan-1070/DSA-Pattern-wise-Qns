# Contribution Guide

Welcome to the DSA Pattern-wise repository! Here is how we will collaborate:

## 📂 Folder Structure
Each pattern folder contains a `.gitkeep` file to keep it active. Add your solutions inside the relevant pattern folder.

## 🐍 Python Solutions
- Save your files as `ProblemName.py`.
- Example: `01-Two-Pointers/TwoSum.py`

## ☕ Java Solutions
- Save your files as `ProblemName.java`.
- Example: `01-Two-Pointers/TwoSum.java`

## 🚀 How to collaborate
1. **Clone the repo** (if you haven't): 
   `git clone https://github.com/Sohan-1070/DSA-Pattern-wise-Qns.git`
2. **Always Pull before working**: 
   `git pull origin main`
3. **Add your code**: 
   Place your `.java` or `.py` file in the correct pattern folder.
4. **Commit and Push**:
   `git add .`
   `git commit -m "Add solution for [Problem Name] in [Language]"`
   `git push origin main`
